# go_auth_serivce
A small project that is a mock authentication service that can create and store users, store passwords, and validate users.
This will interface with an SQL server with a database called authproject.  You can run the create-table.sql to make the needed table. 
You will need to create system environment variables for your SQL username and password user DBUSER and DBPASS to interface with the database.
This is a big work in progress right now and will be updating it until all features are added.
